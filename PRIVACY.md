# Privacy Policy

**Last Updated:** July 2026

## Overview

InsightIQ is an AI-powered browser extension that helps users summarize webpages, answer questions about page content, and analyze user-uploaded files.

## Information We Collect

InsightIQ does not collect or sell personal information.

The extension may access:

- The content of the current webpage selected by the user.
- Files explicitly uploaded by the user for analysis.
- User preferences and settings stored locally.
- API keys provided by the user, which are stored locally in the browser.

## How We Use Information

The accessed information is used only to:

- Summarize webpage content.
- Answer user questions.
- Analyze uploaded files.
- Provide AI-powered assistance.

## Data Storage

User settings and chat history are stored locally in the browser unless the user removes them.

## Third-Party Services

InsightIQ may communicate with AI providers such as Groq, Gemini, or Pollinations to process user requests. Only the information necessary to complete the requested task is sent.

## Data Sharing

InsightIQ does not sell or transfer user data to third parties.
